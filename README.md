# Meridian

Visualize time across multiple timezones with an interactive, draggable timeline.

Meridian is a web application designed to help you easily coordinate across different timezones. It features an aesthetic, clean interface with a draggable timeline that lets you instantly see time conversions and compare working hours globally.

## Features

- **Interactive Timeline**: Drag across the timeline to view future or past times across all selected timezones simultaneously.
- **Multiple Timezones**: Add as many timezones or cities as you need using the built-in search functionality.
- **Working Hours Highlight**: Easily identify overlapping working hours (9 AM – 5 PM) to find the perfect meeting time.
- **Current Time Feature**: Jump back to the current time with a simple click.
- **Modern UI**: Clean and minimal interface optimized for both desktop and mobile devices.

## Setup & Local Development

This project is built using [Vite](https://vitejs.dev/) and [Luxon](https://moment.github.io/luxon/) for timezone calculations.

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Start the development server**:
   ```bash
   npm run dev
   ```
3. **Build for production**:
   ```bash
   npm run build
   ```

## Technologies

- **Vite**: Next Generation Frontend Tooling.
- **Luxon**: A powerful, modern, and friendly wrapper for Javascript dates and times.
- Vanilla JavaScript, HTML, and CSS.
